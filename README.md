The Vortex: A Personal Website

✨ Overview

    This project is my attempt to build a digital space that holds everything I am and everything I create—not just a portfolio, but a living, evolving archive.

    Inspired by early 2000s indie websites, this platform embraces personality over perfection. It is intentionally raw, expressive, and deeply personal—less “look at my achievements” and more “this is my universe.”

    This project is being developed as part of my web development coursework , incorporating JavaScript, API integration, and interactive UI design.

🧠 Concept:
Most websites are curated. This one is documented.

    Instead of hiding the process, this site shows:

    the messy middle
    the unfinished ideas
    the behind-the-scenes of becoming

    Think of it as:
    part notebook, part lab, part gallery, part existential crisis (but aesthetic)

🗂️ Structure:
    🏠 Home:
        The entry point to my world.
        A snapshot of who I am right now—not a static bio, but a moving state.

    📓 Notebook:
        A curated collection of my writings:
            thoughts
            essays
            fragments
            ideas I want others to read
        This is where my brain leaks onto the internet.

    🔬 Projects:
        A transparent log of my academic journey:
            Project topics
            Product breakdowns
            behind-the-scenes (BTS) of how I approach problems
            personal research journal entries

        Not just what I learn, but how I think.

    🚀 The “Impossible” List:
        A public bucket list of things I want to achieve.
        Each item gets crossed off as I complete it.

        It’s equal parts:

            accountability system
            motivation engine
            proof that I actually follow through

    🎨 Portfolio (Art & Projects):
        A space for everything I create:
            art
            side projects
            experiments
            things that don’t fit in neat categories
        No pressure to be “professional”—just honest work.

    🙋‍♀️ About:
        A more personal look at who I am, what I care about, and what I’m trying to build (both externally and internally).

⚙️ Features:
    🌐 Dynamic content rendering using JavaScript
    🔍 Search, filtering, and sorting (using array higher-order functions)
    🎨 Custom UI inspired by retro indie web design
    📱 Fully responsive design
    🌗 (Planned) Dark/light mode toggle
    ⭐ (Planned) Local storage for preferences and interactions

🛠️ Tech Stack:
    HTML5 — structure
    CSS3 — styling and layout
    JavaScript (ES6+) — interactivity and logic
    Public APIs — for dynamic data integration
    (Optional) Tailwind / Bootstrap

This project is not just about completing an assignment. It’s about building:
    a digital identity that evolves
    a public record of growth
    a space where process > polish
